# Roadmap

Visionary Shield grows best from real server feedback. This roadmap is intentionally practical.

## Short term

- More community translations
- More screenshot-basic examples
- Improved setup examples for different QBCore stacks
- Small UI accessibility improvements
- Better troubleshooting docs based on real reports

## Mid term

- Optional exports for third-party admin resources
- More granular Discord routing examples
- Additional moderation workflow presets
- Compatibility notes for common FiveM hosting setups

## Long term

- Community-maintained detection presets
- Safer test harnesses for server owners
- More transparent performance benchmarks

## Philosophy

No black-box promises. No magic claims. The goal is practical visibility, staff-friendly moderation and transparent security tooling.
